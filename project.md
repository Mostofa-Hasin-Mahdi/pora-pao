Key Features:

# Multi login- Sign up only for tutors,
Tutors add their name and number while signing up 
Tutors can create students- student creation leads to generating a code
    which is unique for the student/parent to access
when a student/parent logs in they can click on tutor info and see the tutor's name and number

# Within each student, tutor selects how many days per week he/she 
    teaches the student
this will generate the total number of tution days within that month
after this the tutor can also schedule the timing of the day he/she 
    will be taught
if the tutor can't teach on the tuition day, the tutor can reschedule
    that tuition day to another day, this will pop up as an alert on the
    student/parent dashboard

# Tutors can add subjects they teach
    tutors can also assign homeworks to each subject
to assign a homework, the teacher selects the subject, writes the home work in a text field and submits,
this pops up in the student dashboard
homeworks will have complete, incomplete, partially complete statuses which is set by the tutor for each homework according to the tutor's satisfaction

# Tutors can add quizes, 
tutors can add quizes by selecting the subject and writing on the text field about the syllabus of the quiz
this pops up on the student dashboard as quiz alert

# Student performance tab
this tab can be accessed by both tutor and student/parent 

the performance tab will have the 2 charts 
1. Performance 
this chart shows which subject the student always achieves the best score 
The Look: A hexagon or octagon where each point represents a subject (Math, English, etc.).
Mobile Tip: Keep the labels short (e.g., "Phy" instead of "Physics") to prevent text overlapping on small screens.

2. Subject strength 
this will show charts for the subjects the tutor teaches 
The Look: A ine chart that shows percentage of each quizes (on x axis quiz number, y axis number obtained).

3. Daily performance
this will be based on homework statuses
The Look: A smooth line chart showing the average of all statuses (completed- best score, incompleted- bad score, partially complete- medium score).
Make it mobile responsive

# the tutor and student/parent dashboard will have buttons to show quiz history to see the marks of previous quizes,  and a button to see previous homeworks
